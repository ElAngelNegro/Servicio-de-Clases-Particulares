Descripción del proyecto:

"Servicio de reforzamiento académico para estudiantes de educación media."

Consiste en un landing page que permite a los estudiantes acceder a servicios de reforzamiento académico, con información sobre los cursos disponibles, precios y promociones.

Problemática abordada.

- Nos enfocamos a estudiantes de educación media que necesitan apoyo académico adicional, pero que no tienen los recursos económicos necesarios para acceder a servicios de reforzamiento, con enfoques para la educación superior, a su ves, con fines para que en futuro estén preparados para enfrentar los desafíos de la misma.

Tecnologías utilizadas:

- HTML
- CSS
- JavaScript
- Bootstrap

Explicación breve del uso de IA:

- Se procedió a utilizar GitHub Copilot para la generación de código JavaScript, en lo funcional, animaciones y responsividad, y para la generación de código HTML y CSS, en lo estético, estructura y diseño, facilitando la creación de componentes interactivos y la optimización del código.
